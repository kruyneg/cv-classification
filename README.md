# CV Classification

Исследование и реализация моделей компьютерного зрения для задачи бинарной классификации изображений: различие дронов и птиц.

## Датасет

Используется датасет [Drone vs Bird](https://www.kaggle.com/datasets/muhammadsaoodsarwar/drone-vs-bird). Задача актуальна для систем видеонаблюдения, авиационной безопасности и мониторинга природы, где требуется надёжно отличать потенциально опасные объекты от естественных.

## Запуск

Для запуска ноутбука использовалась среда Google Colab с GPU T4.

Для воспроизведения результатов необходим Python и установленные зависимости:

```bash
pip install torch torchvision scikit-learn matplotlib pandas kagglehub
```

---

## Результаты

В качестве базовых моделей использовались предобученные архитектуры из `torchvision`, дообученные на целевом датасете. Дальнейшие эксперименты включали усиленные аугментации, подбор гиперпараметров и тестирование современных архитектур. Лучшие достигнутые значения *F1-Score*:

| Модель          | *F1-Score* |
|-----------------|------------|
| ResNet18        | 0.9971     |
| ViT-B/16        | 0.9941     |
| EfficientNet-B0 | 0.9990     |

## Собственная модель

Помимо предобученных решений, с нуля спроектирована архитектура `DroneBirdNet`, учитывающая специфику задачи. После обучения этой моделью достигнут *F1-Score* - результат подтверждает работоспособность архитектуры, но ожидаемо уступает моделям, предобученным на больших датасетах.

---

## Вывод

В ходе работы удалось построить модель классификации, достигающую практически идеальной точность (F1 = 0.999) на тестовой выборке, что подтверждает возможность надёжного решения задачи различения дронов и птиц с использованием современных предобученных архитектур. Самостоятельная имплементация модели продемонстрировала работоспособность, хотя и не превзошла другие модели.
